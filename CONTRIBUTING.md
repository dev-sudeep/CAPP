# Contributing to CAPP

Thanks for your interest in contributing! The process is simple: fork the repo, make your changes, and open a pull request.

## Steps

1. **Fork the repository**
   - Click **Fork** on GitHub to create your copy.

2. **Clone your fork**
   ```sh
   git clone https://github.com/<your-username>/CAPP.git
   cd CAPP
   ```

3. **Create a feature branch**
   ```sh
   git checkout -b my-change
   ```

4. **Make your changes**
   - Keep changes focused and update documentation if needed.

5. **Build/test when applicable**
   ```sh
   gcc -o capp capp.c
   ```

6. **Commit your work**
   ```sh
   git add .
   git commit -m "Describe your change"
   ```

7. **Push to your fork**
   ```sh
   git push origin my-change
   ```

8. **Open a pull request**
   - Go to your fork on GitHub and open a PR against the upstream repository on the **main** branch.
   - Provide a clear summary and any relevant details.

## Questions

If you have questions, open an issue with details about what you’re trying to do.
